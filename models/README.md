# Models

Download the YOLOv8-Face model:

## Windows PowerShell
```powershell
Invoke-WebRequest -Uri "https://github.com/akanametov/yolo-face/releases/download/v0.0.0/yolov8n-face.pt" -OutFile "yolov8n-face.pt"
```

## Linux/Mac
```bash
curl -L -o yolov8n-face.pt https://github.com/akanametov/yolo-face/releases/download/v0.0.0/yolov8n-face.pt
```

Or download manually from: https://github.com/akanametov/yolo-face/releases
